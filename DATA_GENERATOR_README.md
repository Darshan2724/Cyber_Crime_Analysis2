## Quick Start

### Step 1: Run the Generator

Open your terminal in the project folder and run:

```bash
python generate_expanded_data.py
```

### Step 2: What It Does

The script will:
- Generate **10,000 realistic cyberattack records**
- Create varied data across 45+ countries
- Include 15 different attack types
- Span from 2019 to 2024
- Save as `expanded_cyber_attacks.csv`

### Step 3: Use the New Data

**Option A: Replace Existing Data**
1. Backup your current CSV file
2. Rename `expanded_cyber_attacks.csv` to your original filename
3. Restart Streamlit app

**Option B: Update Code**
1. In `modules/data_adapter.py`, update the CSV filename
2. Point to `expanded_cyber_attacks.csv`
3. Restart Streamlit app

## Customization

### Change Number of Records

Edit line 15 in `generate_expanded_data.py`:

```python
NUM_RECORDS = 10000  # Change this number
```

### Change Date Range

Edit lines 16-17:

```python
START_DATE = datetime(2019, 1, 1)  # Start date
END_DATE = datetime(2024, 12, 31)  # End date
```

### Add More Countries

Edit the `COUNTRIES` list (line 20):

```python
COUNTRIES = [
    'USA', 'UK', 'Germany',
    'YourCountry',  # Add here
    # ...
]
```

### Add More Attack Types

Edit the `ATTACK_TYPES` list (line 30):

```python
ATTACK_TYPES = [
    'Malware', 'Ransomware',
    'Your Attack Type',  # Add here
    # ...
]
```

## Troubleshooting

### Error: "pandas not found"

Install pandas:
```bash
pip install pandas numpy
```

### Error: "Permission denied"

Run as administrator or check file permissions.

### Need Different Data Distribution?

Edit the `generate_attack_record()` function in the script to adjust:
- Severity ranges
- Financial loss calculations  
- User count multipliers
- Data loss amounts
